# version examples

[This folder](.) contains version examples used by [Reference Guide](https://gammasoft71.github.io/xtd/reference_guides/latest/) and more.

* [version](version/README.md) demonstrates the [xtd::version](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1version.html) constructor, and major, minor, build, revision, major_revision, and minor_revision properties.
* [version_os](version_os/README.md) shows how to use [xtd::version](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1version.html) to get the Operating System version.

## Build and run any project

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```shell
xtdc run -t any_project_name
```
