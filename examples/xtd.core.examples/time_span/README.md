* [time_span_from_days](time_span_from_days/README.md) demonstrates the [xtd::time_span::from_days](https://gammasoft71.github.io/xtd/reference_guides/latest/structxtd_1_1time__span.html#a8417d7db7bed884f491005bccf2bb786) method.
