# Fonts examples

[This folder](.) contains fonts examples.

* [font](font/README.md) Shows how to use [xtd::drawing::font](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1drawing_1_1font.html) class.
* [system_fonts](system_fonts/README.md) Shows how to use [xtd::drawing::font](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1drawing_1_1system__fonts.html) class.

## Build and run any project

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```shell
xtdc run -t any_project_name
```
